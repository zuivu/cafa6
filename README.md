## Create an virtual environment

- Create a new virtual environment if not existed yet:  
  `python -m venv venv`

- Activate the environment:
  - Windows:  
   `venv\Scripts\activate` 
  - Macs/Linux:  
  `source venv/bin/activate` 

- Install dependencies:  
`pip install fair-esm`
